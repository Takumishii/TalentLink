# TalentLink
Sistema de practicas
Modulos Bakend (Laravel)
Geraldine: Core y API
  Gestión de Datos: Crear el CRUD para activos fijos (computadoras, maquinaria, mobiliario), técnicos y mantenimientos.
  API para Angular: Desarrollar los endpoints que el frontend consumirá para mostrar el Dashboard y las tablas.
  Seguridad y Configuración: Implementar el login, 2FA, expiración de sesión y configuración de SMTP para correos.

Patricio: Automatización y Procesos
  Lógica de Fechas: Crear la función para calcular las fechas próximas de mantenimiento.
  Task Scheduling (Cron Jobs): Configurar las tareas automáticas para revisar mantenimientos cada lunes.
  Sistema de Alertas: Programar el envío automático de correos a los técnicos y alertas de vencimiento.
  Base de Datos: Configurar los respaldos automáticos diarios.

Modulo Frontend (Angular)
Bastian: Creacion de interfaces que consumen la lógica de Laravel.
  Dashboard Visual: Crear los indicadores (KPIs) y el gráfico de dona de "Distribución de Mantenimientos".
  Gestión de Estados (Semáforo): Implementar la lógica visual para mostrar equipos en rojo (vencido), amarillo (próximo) y verde (al día).
  Vistas de Usuario:
    Vista Admin: Formularios para crear activos y asignar técnicos.
    Vista Técnico: Tablero de tareas (Por hacer, En progreso, Completado).
  Calendario y Reportes: Implementar la vista de calendario mensual y la sección de analíticas con exportación a PDF/Excel.
